# NotesApp
 Simple NotesApp using FileManager and UserDefaults


### NotesApp Contain Functionality

- Login / Logout (with UserDefaults)
- Add new notes
- Modify existing note
- Delete notes

#### UIElements
 
 - UILabel
 - UIButton
 - UIImageView
 - UIAlertView
 - UITextField
 - UITextView
 - UIButton
 - UITableView
